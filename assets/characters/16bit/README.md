# HIVE 16-bit Character Assets

## Purpose

This directory contains repo-native pixel-art assets for Mobius HIVE.

The goal is not merely retro aesthetics.

These sprites function as:
- civilization archetypes
- governance classes
- Agent embodiments
- and future playable shell entities.

## Art Direction

The HIVE visual direction combines:
- 16-bit RPG readability
- constitutional civilization symbolism
- sacred architecture motifs
- governance iconography
- and cyber-mythic atmosphere.

The art should feel:
- symbolic
- memorable
- readable at small scales
- and tied to institutional roles.
