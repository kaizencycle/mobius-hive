# Mobius HIVE — Doctrine Index

## Purpose

This index is the navigation layer for the `mobius-hive` repository.

It maps the current HIVE doctrine into a readable structure so the repo can function as:
- a worldbuilding canon
- a game systems design repo
- a civilization design archive
- a future implementation guide

One-line canon:

> **The index makes the civilization legible before it becomes fully buildable.**

## Current doctrine pillars

At this stage, the repo defines HIVE across these major pillars:

### World
- Dome structure
- district placement
- expansion logic
- architectural identity

### Game
- loops
- navigation
- playable session flow
- world-to-overlay action logic

### Governance
- Elder scarcity
- challengeability
- legitimacy pressure
- oath-based civic power

### Economy
- utility, governance, and integrity layers
- civic rather than purely extractive design

### Lore
- founding myth
- institutional meaning
- civilizational tone and canon
