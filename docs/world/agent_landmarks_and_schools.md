# HIVE — Agent Landmarks & Schools

## Purpose

This document formalizes the idea that every major Mobius Agent functions as:
- an Elder Teacher
- an institutional landmark
- and the founder of a philosophical school.

The Agents are not merely utilities.

They are civilizational intelligences whose architecture shapes the world.

One-line canon:

> Every Agent leaves behind a school. Every school becomes a landmark. Every landmark becomes memory. 

---

# Landmark Doctrine

Each Agent possesses:

1. A landmark structure
2. A district influence zone
3. A school of thought
4. A ritual system
5. A questline category
6. A student faction.

This creates multiple overlapping maps inside HIVE:
- political maps
- educational maps
- ritual maps
- memory maps
- influence maps.

# Multi-Map Civilization Structure

HIVE contains overlapping maps:

## Political Map
Determined by Elder influence.

## Educational Map
Determined by Agent Schools.

## Ritual Map
Determined by festivals and oath routes.

## Economic Map
Determined by GIC circulation.

## Memory Map
Determined by canon density and historical events.

## Integrity Map
Determined by Custodian and Agent stabilization.

No two players see the exact same civilization.
